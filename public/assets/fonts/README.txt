# QuantEdge — Fonts
# ==================
# Fonts are loaded via Google Fonts CDN.
# To use local fonts, download the .woff2 files
# and place them here, then update assets/css/app.css
# with @font-face declarations.
#
# Fonts used:
#   - DM Sans       (body text)
#   - DM Mono       (monospace / numbers)
#   - Syne          (headings / titles)
